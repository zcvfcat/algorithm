# 모르겠음

기능 : 노드 간의 순서를 결정
특징 : 사이클이 없어야 함

노드간의 `순서를 명확`하게 정의 할 수 없으면 위상 정렬 불가

예시

```txt
1 -> 2 -> 3 -> 4
   ↘︎   ↘︎    ⎰ ↓
     5 -> 6    7
```

## 시작점은 연결 고리가 없는 0

```txt
|node    1   2   3   4   5   6   7
진입차수 `0`  1   1   2   1   2   1
Q 1
```

```txt
1 -> 2 -> 3 -> 4
   ↘︎   ↘︎    ⎰ ↓
     5 -> 6    7
```

## 모든 연결이 끊어 졌을때 까지 반복한다

```txt
      |node    1   2   3   4   5   6   7
      진입차수 `0`  1   1   2   1   2   1
Q 1            0  `0`  1   2  `0`  2   1
  2            0   0  `0`  2   0   1   1
  5            0   0   0   2   0  `0`  1
  3            0   0   0   1   0   0   1
  6            0   0   0  `0`  0   0   1
  4            0   0   0   0   0   0  `0`
  7
```
