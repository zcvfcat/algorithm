# 사용법

## 최적의 해를 찾을 수 있는지 정당성 검토

### 동전 문제

동전이 500, 400, 100인 경우

800원을 거슬러야할 경우  
500 + 100 \* 3 : 400 \* 2 = 4 : 2 이기 때문에 400원으로 거슬려줘야한다.

해당 문제는 동전의 모든 경우의 수를 전부 파악해야한다. -> 다이나믹 프로그래밍으로 해결

하지만 동전이 500, 250, 100인 경우

동전의 집합이 전부 500의 약수의 집합이기에 `집합(500) = {x | x는 500의 배수}`

큰 단위부터 거스름돈을 주면 해결이 가능하다 -> 그리디 알고리즘 적용가능
